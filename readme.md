**Overview:**
---

This project was to develop a PHP based shopping cart application built using **MySQL Database** and a simple **MVC (Model, View, Controller)** structure.
The app displays a list of products that inserted into the MySQL Database to be presented in a PHP based web interface. The cart has add and remove functionality and a 
"_View Cart_" page that calculates cart toals and associated tax and shipping costs.

---
**Project Features:**
- Displays products from a MySQL table titled '_products_'
- Adds products to a cart using PHP sessions
- View Cart page that:
  - Presents all items added to cart
  - Subtotal of all items in the cart
  - Tax amount
  - Shipping costs
  - Grand total (Subtotal + Tax + Shipping)
- CSS Stylesheet for a simple and readable view
- Organized with a MVC structure, enforcing readability, clarity, and reusability
